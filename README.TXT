IT'S VERY IMPORTANT!

Для правильной работы звуков необходимо расположить файл тетриса и папку со звуками в одну папку. Т.е, вы можете создать на рабочем столе папку "TETRIS" и положить туда 2 файла Tetris.jar && tetrissounds. При правильном выполнении условии, вы получите правильные звуки.
